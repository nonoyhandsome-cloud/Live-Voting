# 🚀 Setup and Run the Live Voting App
1. Setup: Place index.html, dashboard.html, voting.js, dashboard.js, and style.css in one folder.

2. Launch: Open index.html in any browser or use VS Code with the provided launch.json.

3. Vote: Enter a unique ID, select one candidate for each of the five positions, and click Submit Vote.

4. Results: Click Go to Dashboard or open dashboard.html to view the live tallies.

5. Reset (Optional): To clear all data and restart the election, open your browser's Developer Tools (F12), go to Application > Local Storage, and delete the electionData key.
